FC-Taylor program

Calibrate a yield surface for an FCC polycrystal

Home

This program uses the full-constraint (FC) Taylor homogenization approach to calculate the discrete yield surface of an FCC polycrystal. The Yld2004-18p yield surface can be fitted to the discrete points calculated.

How to

  • Select a file containing information about the crystallographic texture of the material under the "Texture" tab.
  • Input your material parameters under the "Material" tab.
  • Setup and start the FC-Taylor job under the "Job" tab.
  • Monitor the progress under the "Monitor" tab.
  • When the job has finished, you can save the discrete yield surface data points under the "Job" tab by pressing "Save results".

Calibration and visualization

The discrete yield surface data points can be used to calibrate the Yld2004-18p yield surface for the material.

When the job has finished, you can do the following:

  • Go to the "Calibrate" tab and press "Start calibration" to calibrate the Yld2004-18p yield surface.
  • Go to the "Visualize" tab to plot the discrete yield surface points and the calibrated yield surface.

Texture format

The texture file should contain the keyword *Euler at the top of the file, followed by lines containing Euler angles (in degrees) and an orientation weight separated by commas , i.e., phi1, PHI, phi2, weight. Lines starting with ** are treated as commented lines and thus not read.

A typical texture file can look like this:

*EULER ** This is a commented line **phi1, PHI, phi2, weight - This is also a commented line 51.07, 28.75, 344.14, 7.500 33.18, 43.11, 5.45, 4.500 44.53, 32.55, 349.77, 6.500 39.81, 38.04, 354.51, 5.000 29.26, 41.79, 7.10, 2.000 223.19, 38.99, 167.21, 3.500 231.21, 35.43, 159.54, 8.500 45.01, 31.46, 352.26, 0.420 123.45, 32.27, 198.15, 4.000 227.21, 43.44, 170.03, 1.500
